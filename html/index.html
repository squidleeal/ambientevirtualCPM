Este Projeto foi feito em parceria com a Universidade Federal do Tocantins, com o projeto Conecte-se, os alunos da UFT se mobilizaram para levar um conhecimento básico de criação de sites, utilizando HTML, CSS. Este projeto é um exemplo do que essas linguagens podem fazer, e como elas podem ser utilizadas para criar um site responsivo, que se adapta a qualquer tela, seja de celular, tablet ou computador.
